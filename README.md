# Sistema de Registro para Eventos

Sistema CRUD para registro de participantes con folios únicos.

## Características
- Registro con folio único (EVT-XXXXXX)
- Búsqueda por folio
- Listado de registros
- Eliminación de registros
- Diseño responsive

## Tecnologías
- PHP 8+
- MySQL
- HTML5, CSS3, JavaScript
- Railway para deployment

## Despliegue en Railway
1. Conectar repositorio a Railway
2. Añadir base de datos MySQL
3. La aplicación se despliega automáticamente
